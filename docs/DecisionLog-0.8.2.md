# Decision Log 0.8.2

## DL-012 — Dashboard derivata dalle API esistenti
La dashboard 0.8.2 non introduce nuove tabelle né nuove API. I KPI sono calcolati usando gli endpoint Admin già disponibili.

## DL-013 — Copertura di mappatura
La copertura è un indicatore descrittivo calcolato come articoli associati / articoli attivi. Non rappresenta il completamento della zona e non introduce workflow forzati.

## DL-014 — Aggiornamento automatico
La dashboard si aggiorna ogni 30 secondi solo quando la pagina è visibile. L'aggiornamento può essere richiesto anche manualmente.
