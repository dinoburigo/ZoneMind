# Decisioni Sprint 0.7.0

## DL-016 - Admin come punto di gestione dei dati

La console Admin diventa il punto unico per import catalogo, pubblicazione layout e consultazione dello stato centralizzato.

## DL-017 - Dashboard basata su dati oggettivi

La dashboard mostra esclusivamente quantità calcolate dal database: articoli attivi, EAN, zone e associazioni.

## DL-018 - Migrazione incrementale del database

Il database SQLite locale viene aggiornato senza cancellare i dati già acquisiti.

## DL-019 - Un solo layout attivo per negozio

La pubblicazione di un nuovo layout disattiva il precedente per lo stesso negozio.
