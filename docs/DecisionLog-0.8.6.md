# Decision Log 0.8.6

## DL-018 – Origine delle associazioni
Le associazioni ufficiali articolo-zona sono create esclusivamente dal Mapper tramite scansione di un barcode valido per il catalogo del negozio.

## DL-019 – Admin Console in sola lettura
Assignment Explorer è uno strumento di consultazione, ricerca, audit ed esportazione. Non crea, sposta o elimina associazioni.

## DL-020 – Manutenzione rinviata
Le modifiche amministrative saranno introdotte solo con autenticazione, ruoli e audit completo.
