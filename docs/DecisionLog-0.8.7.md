# Decision Log 0.8.7

## DL-020 – System Console non distruttiva
La System Console espone diagnostica, versioni, integrità e volumi senza modificare dati o configurazione.

Con Dashboard, Negozi, Catalogo, Layout, Associazioni e Sistema, la copertura funzionale della Admin Console è completa per la release 0.8.x.
