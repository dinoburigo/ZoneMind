# ZoneMind 0.3.0 - Minimal Zone Editor

**Data:** 13/07/2026  
**Stato:** Conclusa

## Obiettivo

Realizzare un Editor delle zone rapido e intuitivo, progettato per richiedere il minimo effort operativo.

## Funzionalità introdotte

- caricamento di una planimetria;
- modalità di disegno continuo;
- creazione delle zone mediante trascinamento;
- assegnazione automatica dei codici A01, A02, A03...;
- assegnazione automatica del colore;
- selezione di una zona;
- eliminazione della zona selezionata;
- uscita esplicita dalla modalità di disegno;
- esportazione del layout in formato JSON;
- importazione del layout completo;
- inclusione temporanea dell’immagine della planimetria nel JSON Base64.

## Flusso operativo

1. Aprire la planimetria.
2. Premere **Nuova zona**.
3. Trascinare il mouse per definire la zona.
4. Il sistema assegna automaticamente il codice.
5. Continuare a disegnare le zone successive.
6. Premere **Fine editing**.
7. Se necessario, selezionare ed eliminare una zona.
8. Esportare il layout.

## Regole UX

- nessun nome descrittivo obbligatorio;
- nessun pannello proprietà;
- nessun ridimensionamento;
- nessun `prompt`;
- nessuna conferma per la creazione;
- modifica tramite eliminazione e ricreazione;
- modalità semplice predefinita;
- funzionalità avanzate rinviate a configurazioni opzionali.
