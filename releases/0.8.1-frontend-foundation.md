# ZoneMind 0.8.1 - Frontend Foundation

## Contenuto
- nuova shell Admin con sidebar e header responsive;
- moduli comuni per API, formattazione, toast e navigazione;
- grafica Admin riorganizzata;
- mantenimento di tutte le funzioni della 0.7;
- preparazione delle future sezioni Negozi, Catalogo, Layout, Associazioni e Sistema.

## Compatibilità
Compatibile con il backend 0.7.x e con il database esistente. Nessuna migrazione dati richiesta.
