# ZoneMind 0.7.0 - Admin Data Management

## Obiettivo

Trasformare la pagina di import in una console amministrativa minima per la demo.

## Funzionalità

- selezione negozio;
- stato API;
- KPI catalogo, EAN, zone e associazioni;
- ultimo import CSV;
- import catalogo con riepilogo;
- pubblicazione layout JSON;
- anteprima ricercabile degli articoli;
- consultazione delle associazioni centralizzate;
- migrazione automatica del database SQLite esistente.

## Fuori scope

- autenticazione e ruoli;
- modifica manuale degli articoli;
- cancellazione massiva;
- storico layout completo;
- immagini prodotto;
- scheduler automatico degli import.
