# ZoneMind 0.5.0

## Installazione

Copiare i file nelle seguenti posizioni del repository:

- `public/index.html`
- `public/assets/js/mapper.js`
- `public/assets/css/mapper.css`

## Test rapido

1. Aprire il Mapper.
2. Verificare il caricamento della planimetria.
3. Toccare una zona sulla planimetria.
4. Verificare l'apertura dello scanner.
5. Leggere uno o più barcode.
6. Premere **Termina lettura**.
7. Verificare il conteggio aggiornato nella planimetria e nel riepilogo.
8. Toccare una zona dal riepilogo e verificare che lo scanner non parta.
